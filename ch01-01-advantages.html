Concise Rust

Advantages

Modern

  • Typed language
  • General purpose
  • Explicit

Safe and Memory efficient

  • Ownership and Borrowing
  • No Garbage collector
  • Fast (like C/C++)

Type inference


#![allow(unused)]
fn main() {
let str = "I am not really a String ;-)"; // -> &str
let val = 40_000 // -> i32
let val = 40. // -> f64
}

Immutable by default


#![allow(unused)]
fn main() {
let foo = 32;
foo = 40; // Nope
}

Let's make it mutable.


#![allow(unused)]
fn main() {
let mut foo = 32;
foo = 40;
}

Variable names are reusable


#![allow(unused)]
fn main() {
let x = 32;
let x = 32 + 8; // still immutable!

println!("{}", x); // Only the last x will be available
}
